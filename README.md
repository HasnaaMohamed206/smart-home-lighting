# Smart Lighting Home

An intelligent Smart Lighting Home system developed using Prolog (TUe Prolog) and integrated with a JavaScript UI.

## Project Structure
- `src/` : Prolog logic, rules, and JavaScript UI code
- `doc/` : Project documentation and reports

## Technologies Used
- Prolog (TUe Prolog)
- JavaScript

## Description
This project simulates a smart home lighting system using rule-based reasoning to automatically control lights based on different conditions.

## How to Run
1. Open the project folder using VS Code.
2. Open the file `samrt-home.pl.js`.
3. Make sure you are connected to the internet (required for external libraries/links).
4. Run the file using a live server or open it in the browser.
5. Interact with the Smart Lighting system through the UI.
